## Sorting Javascript

this is simple package to implementation alogrithm sorting in javascript

## How To Use ?

Use NPM

```
    npm install sorting-js
```

## Algorithm Sorting List

- Bubble Sort(✔)
- Insertion Sort(✔)
- Selection Sort(✔)
- Merge Sort(⏳)
- Radix Sort(⏳)
- Shell Sort(✔)
- Cycle Sort(✔)
